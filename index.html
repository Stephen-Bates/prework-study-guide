HTML

  • The <head> element contains information about the webpage.
  • The <link> element associates the HTML with an external file.
  • The <body> element represents the visible content shown to the user.
  • The <ul> element creates an unordered list, and <li> elements add to that list.
  • Element attributes are defined inside the opening tag and provide additional information about the element.
  • The <a> element creates a hyperlink.
    • The href attribute specifies the URL of the link
  • The <img> element adds an image
    • the src attribute specifies the path to the image
    • the alt attribute gives the image an alternate text to be displayed if unable load the image
  • To add certain characters as text, they must first be escaped to prevent them from being interpereted as code
    • < = &lt;
    • > = &gt;
    • & = &amp;
    • " = &quot;
    • ' = &#39;

CSS

  • Rules are created by declaring the elements type they apply to, followed by the attributes and values encapsulated in curly braces
  • multiple element types can be grouped to the same rule, seperated by commas
  • A class element is denoted by a period followed by the name of the class. (.example{})
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Colors can be specified by name or through RGB hex values. (#444444)

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: updates current branch with changes to the main branch
  • git add [file]: stages changes to the specified file for next commit. use -A flag to add all changes
  • git commit -m "[descriptive message]": commits the staged changes
  • git push origin [branch-name]: transmits local branch commits to the remote repository branch

JavaScript

  • A variable is a named container that allows the storage of data in our code.
  • A list starts at index of 0, and it's items can be access by the list name proceeded by '[i]', where i is the index of the desired value
  • Contorl flow is the order in which a computer executes code in a script.
